Mandelbrot

Background

The mandelbrot set is the set of values for c for which z → z² + c is bounded, where z starts at zero and c is any complex number. Somehow, this simple definition produces staggering complexity.

This application visualizes the mandelbrot set by associating every pixel with a value of c, using the x and y coordinates of the pixel to derive the real and imaginary parts of the complex number. Members of the mandelbrot set are rendered black, and non-members are rendered a color determined by how quickly z exceeded a bound of 2 (if it exceeds a bound of 2, it can be proven that it will exceed any bound).
